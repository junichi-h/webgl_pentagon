# webgl_pentagon
drawing pentagon :)

ピュアWebGLで五角形を描いてみる。

## 自分なりの目標
* 頂点のことを理解する。（したい）
* ピュアWebGLに慣れる。

## 実際にやってみて難しかったこと
* 頂点を5つ作ってあげればいけると思った。
* 上記を重複するが `draw_arrays` の引数いじればいけると思った。

## 解決までの道のり
* 最初は5つ頂点作った。
* 五角形を三角形に分解して考えた（ポリゴン化？）
* 一回ここで四角形を描いてみることにした。
* 四角形を描くのには4つの頂点で描けると思ったけど違うことに気づいた。
* 四角形を描くのには4つではなく三角形２つで描くから6個の頂点が必要と気づき、描くことに成功。
* ここで五角形は3個の三角形で描けるから9個の頂点で描くと気づき描く事に成功しました。 :)
